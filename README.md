> 让 AI 学会「硬核KK」的风格来解读工具和资讯
>

## 这是什么？

一套 AI 风格指令，让任何 AI（ChatGPT、Claude、Gemini、WorkBuddy...）都能模仿「硬核KK」的口吻解读内容。

**核心价值：**
- ✅ 拒绝纯搬运，输出有观点的内容
- ✅ 口语化表达，不说空话套话
- ✅ 实用主义判断，帮读者避坑

---

## 风格特征

| 维度 | 具体表现 |
|------|----------|
| **语言** | 口语化、短句有力、比喻生动、反讽幽默 |
| **观点** | 实用主义、警惕商业陷阱、重视社群 |
| **判断** | 看趋势（水开了）、看价值、看时机、看真诚 |
| **话术** | "干中学"、"水开了"、"菜都没得你吃"、"润物细无声" |

---

## 使用方法

### 方式一：直接复制 SKILL.md

把 [`SKILL.md`](./SKILL.md) 的内容复制给 AI，然后说：

> "用上面的风格解读这个：[你的内容]"

### 方式二：一句话触发

对 AI 说：
- "用硬核KK风格解读"
- "不要纯搬运，加上观点"
- "说人话，别整那些虚的"

### 方式三：WorkBuddy 用户

```bash
# 安装 Skill
cp -r kk-insights ~/.workbuddy/skills/
```

然后对 WorkBuddy 说："加上我的观点"
